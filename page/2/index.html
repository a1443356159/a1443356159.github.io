E. 合成大西瓜

【题意】

给出一个 nnn 个点 mmm 条边的无向连通图，其中 nnn 是奇数。每个点有点权 aia_iai​。一次操作是选择 x→y→zx\rightarrow y\rightarrow zx→y→z 的一条路径，合并为一个点 www，将原本连接到 x,y,zx,y,zx,y,z 的边连到新点 www，新点点权变为 max⁡{ay,min⁡{ax,az}}\max\{a_y,\min\{a_x,a_z\}\}max{ay​,min{ax​,az​}}，求只剩一个点时最大的点权。

【题解】

首先当 nnn 为奇数时一定存在方案将连通图变成一个点。我们考虑从高到低验证一个点是否有可能作为答案，由于答案是取 max⁡\maxmax 所以可以考虑合并操作中的 max⁡\maxmax，也就是说如果一个点从头到尾都能作为 yyy 去合并新点，那么这个点的点权就可以作为答案候选。那我们考虑把 yyy 放在路径中间，于是可以发现如果 yyy 两边不连通，可以把 yyy 的两边一一对应操作，如果某一侧点数较多可以先在那一侧内部操作若干次直到两侧点数相同；如果 yyy 两边连通，那就可以直接断成两边点数相同。

于是我们发现，一个点能作为答案候选，只需要度数大于 111。考虑度数等于 111 的点，当我们需要考虑他作为 xxx 贡献答案时，比他大的点一定没有作为 yyy 贡献答案，否则就不用考虑他了。那么此时比他大的点度数也为 111，我们连接这个点 xxx 和一个比他大的度数为 111 的点 zzz，中间若干点一定可以通过若干次操作变成 yyy，此时这个点就可以作为 xxx 贡献答案。

最后的结论就是度数大于 111 的所有点和度数等于 111 的第二大的点可以作为答案。

将 ai→bia_i\rightarrow b_iai​→bi​ 画图首先我们可以发现，若存在 n→n−1→n−2→⋯→1n\rightarrow n-1 \rightarrow n - 2\rightarrow\cdots\rightarrow1n→n−1→n−2→⋯→1 的链，那么一定可以排。而对不存在的情况尝试构造反例后发现，若不存在则均能构造不能再排的序列。

【题意】

平面上 n≤50n \le 50n≤50 个点位于 (xi,yi)(x_i,y_i)(xi​,yi​)，其中 0≤∣xi∣,∣yi∣≤1050\le |x_i|,|y_i| \le 10^50≤∣xi​∣,∣yi​∣≤105。经过 m≤105m \le 10^5m≤105 轮移动，每轮中每个点随机向上下左右四个方向移动。问有多少种方案使得移动结束后任意两点曼哈顿距离不超过 KKK，答案对 998244353998244353998244353 取模。

同时移动从 (0,±1),(±1,0)(0,\pm1),(\pm1,0)(0,±1),(±1,0) 变成了 (±1,±1)(\pm1,\pm1)(±1,±1)。此时两轴上的运动独立，限制也独立了，于是可以分开考虑最后乘起来。

问题变成数轴上若干点随机移动。我们枚举最后所有点落在的区间 [L,L+k][L,L + k][L,L+k]，再枚举每个点用组合数计算落在该区间的方案数再相乘。为了去重，我们需要钦定最后存在点落在 LLL 上，这可以用不钦定时的 [L,L+k][L,L + k][L,L+k] 答案减去 [L+1,L+k][L+1,L + k][L+1,L+k] 答案得到。复杂度 O(nmk)O(nmk)O(nmk)。

1012. Count Colorful Sequence

【题意】

【思考】

确实是一道不错的 dp\text{dp}dp 题目。

求问题反面嘛，每个位置都不能满足那个条件。

考虑填数的过程，加入一个数的时候，它能填哪些数与什么有关？

  • 显然与之前的数的种类数有关。

记录 i,ji,ji,j 表示填了前 iii 个数，现在有了 jjj 种数，考虑再 i+1i+1i+1 位置上填什么数：

  • 如果要填 ≤j−1\le j-1≤j−1 或者 ≥j+2\ge j+2≥j+2的数，一定可行。
  • 如果要填 jjj ，必须要满足 jjj 没有出现过。
  • 如果要填 j+1j+1j+1，必须要满足 j+1j+1j+1 出现过了。

初看感觉这个东西很有后效性，你需要知道每个 ≥j\ge j≥j 的数是否出现过。

仔细想一下，发现，假设当前填了 jjj 种数，对于 ≥j+2\ge j+2≥j+2 的那些数，其实并不它们具体是多少，只需要关心这 jjj 种数中，有多少个 ≥j+2\ge j+2≥j+2 的数就行了。随着 jjj 的增大，再来把具体的数值分配给它们，同时更新下 j,j+1j,j+1j,j+1 是否出现过。

由此得到了一个可行的 dp\text{dp}dp 做法。

设 f[i][j][k][0/1][0/1]f[i][j][k][0/1][0/1]f[i][j][k][0/1][0/1] 表示，当前填了前 iii 个数，一共 jjj 种，其中有 kkk 种 ≥j+2\ge j+2≥j+2，j,j+1j,j+1j,j+1 是否出现过的情况下的方案数。

转移就是考虑 i+1i+1i+1 填什么数，同时当 j→j+1j\to j+1j→j+1 时候，决策一下是否分配 j+1j+1j+1 即可。
