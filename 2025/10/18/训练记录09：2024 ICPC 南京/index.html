B.Birthday Gift

【题意】

给出一个长为 n≤2×105n \le 2\times 10^5n≤2×105 的仅由 0,1,20,1,20,1,2 组成的字符串。一次操作可以将一个 222 改为 000 或 111，也可以一次操作将两个相邻的 000 或 111 消掉。问最后剩下的长度最小是多少。

【题解】

值得注意的是，不能 dp 前 i 个字符结尾为 0/10/10/1 的最短长度。因为存在可能需要在前面扩充长度去匹配后面的来消掉。

一种思路是直接维护 0/10/10/1 开头 0/10/10/1 结尾所有可能的长度，不会超过 n2\frac n22n​，用 bitset 可以做到 n22ω\frac {n^2}{2\omega}2ωn2​，但是多组数据 ∑n≤5×105\sum n \le 5\times 10^5∑n≤5×105 大概还是能卡掉。

另一种思路是我们把 222 之间的块分段，每个段内能消的先消，然后就是若干个 0/1…0/10/1\dots0/10/1…0/1 的 010101 交替的串的合并，枚举 222 选择 0/10/10/1，发现两种选择的长度相差只会为 222，考虑可能可以维护 0/1…0/10/1\dots0/10/1…0/1 的可能长度取值区间。

但两段合并还要考虑长度导致的开头结尾变化。沿用这个思路，我们分析一个 0/1…0/10/1 \dots0/10/1…0/1 后面加上单个 0/1/20/1/20/1/2 的可能区间变化。我们发现无论 0/1/20/1/20/1/2 一种一定转移到另一种且是一个置换即不会有两种对应到一种，而且只会有区间左移，右移，左右扩展（即 222）三种变化。所以维护每种类型的可能区间即可。

实现时需要注意区间左端点的特殊情况，可以将单个 0/10/10/1 以及空串是否可能单独列出来，在维护可能区间时注意一下左端点，稍微有点细节。

C. Topology

【题意】

给定一棵树，对节点i在拓扑序中排第i位的方案数。对998244353取模。

n≤5000n \le 5000n≤5000

【解法1】

一个 nnn 个节点的外向树的拓扑排序的数量为 n!∏sizi\frac{n!}{\prod siz_i}∏sizi​n!​。

设 dpx,idp_{x,i}dpx,i​ 表示在点 x 放在位置 i 上，不考虑 x 子树内的方案数。

转移就是 dpy,j=∑i<jdpx,i×(n−i−sizxsizx−1−sizy)×tmpdp_{y,j} = \sum _{i < j}dp_{x,i}\times \binom{n - i - siz_x}{siz_x-1 - siz_y}\times tmpdpy,j​=∑i<j​dpx,i​×(sizx​−1−sizy​n−i−sizx​​)×tmp

tmp指的是x子树内去掉y子树的其他子树的拓扑序方案数。

发现虽然有枚举 i，但是和式里面没有出现 j，所以对于不同的j，相同的i贡献是一样的。

所以可以考虑直接枚举 i 同时枚举 j，也就是利用前缀和优化就可以。

时间复杂度 O(n2)O(n^2)O(n2)。

【解法2】

确定拓扑序的过程一定要是从前往后这，这也是我们dp的顺序，只有这样才能最大化利用dp的状态。

G. Binary Tree

【题意】

交互题。

给你一棵二叉树，你需要在 ⌊log⁡2n⌋\lfloor \log_2 n \rfloor⌊log2​n⌋ 步数限制内确定目标的节点。

每次询问形如，? x y，交互器会返回 0/1/2 ，分别表示：目标节点与 x 相距更近、目标节点与 x,y 距离相等、目标节点与 y 距离更近。

【题解】

在重心处划分即可。

重心至多连接到 333 条边，设连接 x,y,zx,y,zx,y,z 三个点，对应的子树大小为 sx,sy,szsx,sy,szsx,sy,sz 的。

假设其中最小的为 sxsxsx，那么可以通过询问 ? y z 来将树划分为 sx+1,sy,szsx + 1,sy,szsx+1,sy,sz 三部分，容易证明，大小至少变为 ⌊n2⌋\lfloor \frac{n}{2} \rfloor⌊2n​⌋。

实现的代码类似于点分治，考场上代码写得太丑了，导致没有调试通过。

M. Ordainer of Inexorable Judgment

【题意】

从原点引一条已知方向 (x0,y0)(x_0,y_0)(x0​,y0​) 的射线，攻击范围是到这条射线距离小于等于 ddd 的点的集合。射线以 1rad/s 的速度逆时针旋转。给定一个 n≤100n \le 100n≤100 个点凸多边形，保证每个点到原点距离大于 ddd，求 [0,t][0,t][0,t] 时间内攻击范围与凸多边形有交的时间长度。

【题解】

判断攻击范围到达与离开每个顶点的角度，可以求每个点到原点半径为 ddd 的圆的两条切线，两切线一个对应到达一个对应离开，但是要对应到射线的角度还需要经过计算，精度不优。

一个更好的做法是求原点到顶点半径为 ddd 的圆的两条切线，射线碰到圆时攻击范围刚好碰到顶点。

然后由于凸多边形到原点距离都大于 ddd，那么可以发现这个凸多边形一定落在某个半平面内，用顶点坐标找到这个半平面后取出最靠左和最靠右的切线即可。
